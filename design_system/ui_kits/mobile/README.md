# Kilsar Mobile UI Kit

A click-through recreation of the Kilsar mobile app shell using the design system tokens. The kit is **cosmetic** — it shows pixel-accurate components arranged into real screens; it is not production code.

## Files

- `index.html` — runnable click-through prototype (4 screens, swipe between).
- `App.jsx` — top-level shell, navigation state.
- `screens/` — Home, ExamList, Calibration, Profile.
- `components/` — Button, Card, Pill, ListRow, Avatar, BottomNav, TopBar, Input, ProgressBar.
- `IosFrame.jsx` — device bezel (starter component).

Colors, fonts, radii, and spacing all come from `../../colors_and_type.css`.
